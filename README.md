## Data Sources

| Data | Source | Cost |
|---|---|---|
| Map display | [OpenStreetMap](https://www.openstreetmap.org) | Free |
| Route calculation | [OSRM - Open Source Routing Machine](https://router.project-osrm.org) | Free |
| Weather & Rainfall | Mock data based on Kerala monsoon patterns (OpenWeatherMap integration ready) | Free |
| Flood prone zones | Manually curated dataset based on publicly known flood-prone areas in Kerala | Free |

## About
MazhaPath was built as a resume project to demonstrate full stack development
skills using real-world data from Kerala's monsoon season. The name "Mazha"
means rain in Malayalam.
