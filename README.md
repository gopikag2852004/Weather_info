## Weather-info
A monsoon route predictor for Kerala that estimates travel delays, flood risks and safety scores based on rainfall data.

## Data Sources

| Data | Source | 
|---|---|
| Map display | [OpenStreetMap](https://www.openstreetmap.org) |
| Route calculation | [OSRM - Open Source Routing Machine](https://router.project-osrm.org) |
| Weather & Rainfall | Mock data based on Kerala monsoon patterns (OpenWeatherMap integration ready) |
| Flood prone zones | Manually curated dataset based on publicly known flood-prone areas in Kerala |
